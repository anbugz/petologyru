Политика конфиденциальности

Последнее обновление: 1 июля 2026

Бот «Душа в душу» (@SoulPetBot) уважает конфиденциальность пользователей. Этот документ описывает, какие данные мы собираем и как их используем.

Какие данные мы собираем

  • Дата, время и место рождения питомца — хранятся в базе данных для расчёта натальной карты и не передаются третьим лицам.
  • Telegram ID пользователя — используется в анонимизированном виде (необратимое хеширование) для связи данных с аккаунтом внутри бота.
  • История запросов к боту (натальные карты, таро-расклады, гороскопы) — хранится для отображения истории обращений в самом боте.
  • Данные о платежах через Telegram Stars — обрабатываются самой платформой Telegram; мы получаем только факт и сумму оплаты для активации услуги.

Данные второго участника при расчёте совместимости

Если вы запрашиваете совместимость питомца с человеком (например, с собой или членом семьи), дата, время и место рождения этого человека используются один раз — только для выполнения расчёта — и не сохраняются в базе данных после получения результата. В истории обращений остаётся только факт запроса совместимости, без натальных данных человека.

Как мы используем данные

Собранные данные используются исключительно для работы функций бота: расчёта астрологических раскладов, ведения истории обращений, персонализации ответов на основе ранее указанных фактов о питомце.

Мы не передаём данные третьим лицам, не используем их в рекламных целях и не продаём.

Хранение и удаление данных

Данные о питомце хранятся до тех пор, пока пользователь не запросит их удаление. Для удаления всех данных, связанных с вашим аккаунтом, напишите об этом в бот — данные будут удалены безвозвратно.

Обработка через внешние сервисы

Для генерации текстовых разборов бот использует сторонний AI-сервис (DeepSeek). Запросы к нему не содержат вашего Telegram ID или иных идентифицирующих данных — только обезличенные данные, необходимые для расчёта (даты, знаки зодиака, вопросы пользователя).

Контакты

По вопросам, связанным с данными и конфиденциальностью, обращайтесь к @TarologySoul.
